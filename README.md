# 💎 DivineGem – AI-Assisted Gemstone Recommendation Platform

## Overview

DivineGem is a role-based gemstone recommendation and consultation management platform designed to provide users with personalized gemstone guidance based on their zodiac sign and life concerns. The platform also enables users to book consultations and allows administrators to monitor overall platform activity through a dedicated admin dashboard.

The objective of this project was to build a functional Minimum Viable Product (MVP) demonstrating product thinking, clean architecture, and full-stack development skills within a limited timeframe.

---

## Features

### User Features

* User Registration and Authentication
* Secure Login and Logout
* Personalized Gemstone Recommendations
* Recommendations based on:

  * Zodiac Sign
  * Life Concerns (Career, Finance, Health, Education, Love & Marriage)
* Recommendation History Tracking
* Astrologer Consultation Booking
* Modern Dashboard Interface

### Admin Features

* Role-Based Access Control
* Platform Analytics Dashboard
* View Registered Users
* Monitor Recommendation History
* Track Consultation Requests
* Key Platform Metrics

## Installation and Setup

### Clone the repository

```bash
git clone <YOUR_GITHUB_REPOSITORY_URL>
cd divinegem
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```

## User Workflow

1. Create an account or log in.
2. Access the user dashboard.
3. Select zodiac sign and primary concern.
4. Receive a personalized gemstone recommendation.
5. Review recommendation history.
6. Book an astrologer consultation if required.

---

## Admin Workflow

1. Log in using administrator credentials.
2. Access platform analytics.
3. Monitor registered users.
4. Review gemstone recommendations generated by users.
5. Manage consultation requests.
